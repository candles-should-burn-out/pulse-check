Pulse Check

Сервис для аккуратного сбора и просмотра агрегированных показателей без хранения персональных заметок на сервере. Вы фиксируете контекст у себя, а сервер считает только обезличенную статистику.

Возможности Pulse Check

Ведение заметок

Используйте локальную метаинформацию для собственного контекста: она не отправляется и не хранится на сервере.

Агрегированная статистика

Backend получает только данные, нужные для агрегированных показателей, сводок и сравнения состояний.

Командный сбор данных

Назначайте участников, которые помогают собирать показатели, и просматривайте их вклад в своей рабочей области.

Pulse Check отделяет личный контекст от общей статистики.
