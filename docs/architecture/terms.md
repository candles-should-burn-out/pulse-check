## Термины

- Сущность (Entity) - отслеживаемая сущность с server-generated UUID. User-visible names или notes локальны по умолчанию
- Статус (Status) - статус сущности, используемый для локального отображения и агрегированной статистики
- Участник (Participant) - Приглашенный участник, который может участвовать в сборе статистики по статусам (руководителя) без раскрытия конкретных сущностей руководителю
- Пользовательский бэкап (Backup) - Пользовательский зашифрованный export метаданных, который позже можно import
- `entity_id` - технический UUID сущности
- `status_id` - технический идентификатор status
